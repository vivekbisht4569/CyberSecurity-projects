# Password Strength Checker (Python)

This project is a simple Python program that checks the strength of a password entered by the user.  
It analyzes the password based on different conditions such as length, uppercase letters, lowercase letters, numbers, and special characters.

## Features

The program checks whether the password contains:

- Minimum length
- Uppercase letters (A-Z)
- Lowercase letters (a-z)
- Numbers (0-9)
- Special characters (!, @, #, etc.)

Based on these conditions, the password is classified as:

- Strong Password
- Medium Password
- Weak Password

## How It Works

1. The program asks the user to enter a password.
2. It checks every character in the password.
3. It verifies whether the password contains uppercase letters, lowercase letters, numbers, and symbols.
4. Finally, it prints the strength of the password.
